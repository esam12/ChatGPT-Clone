# 🤖 Gemini AI Chat App

A sleek and responsive AI-powered chat app inspired by Google Gemini. Built with Flutter for the client and Node.js for the backend, this app allows users to interact with a generative AI in real time.

---

## 🚀 Features

- Real-time chat interface with streaming responses
- Gemini-like conversational UI
- Message caching and history
- Suggested prompts
- Typing animation for model responses
- Fully responsive and mobile-optimized

---

## 🛠️ Tech Stack

### Client (Flutter)
- Flutter & Dart
- BLoC for state management
- Material Design UI
- REST API Integration

### Server (Node.js)
- Express.js
- TypeScript
- OpenAI / Google Generative AI API (adjustable)
- CORS & body-parser
